Minimum Length

Description

Ensures that the length of a string is greater than or equal to the given value.

Default Message

Minimum length not met. Must be greater than or equal to {length}.

Template Fields

  • {field} - The property name
  • {length} - The minimum length

Usages

  1. public string LongerThanTen { get; set; }
  1. Property(x => x.LongerThanTen).MinimumLength(10);
